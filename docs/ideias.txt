Tema: Editor de texto estilo VIM

1. Possível Inovação:
    * Modo de Ajuda Dinâmico: Um sistema de ajuda integrado que exibe dicas
  contextuais sobre comandos e atalhos à medida que o usuário trabalha, 
  sem sair do editor.
    * Comandos de Acesso Rápido: Implementar um sistema de comandos acessíveis
   com autocomplete para facilitar a navegação e a execução de funções comuns.

2. Edição colaborativa:
    * Sincronização em Tempo Real: Implementar funcionalidades para sincronizar 
    edições entre múltiplos usuários conectados ao mesmo documento via uma 
    conexão de rede (como WebSocket).

3. Modo de navegação:
    * modo de edição: editar o texto escrevendo ":[w]"
    * modo de visualização: visualizar o texto escrevendo ":[v]" e se movimentar com "wasd"
    * modo de comando: inicie qualquer comando com ":[]" dentro dos colchetes você adiciona o comando
